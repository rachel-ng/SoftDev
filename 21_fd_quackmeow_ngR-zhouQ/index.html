Foundation Framework

Foundation is a framework, you can download it here.

File are generally organized like this:

foundation/
  ├─ css/
     ├─ app.css
  ├─ js/
     ├─ app.js
  └─ index.html

We'll be using CDN to access Foundation, so while downloading it isn't necessary, the index.html file in the .zip is a nice example to look at.

Basic HTML Template

This is a super basic template that uses CDN. If you have custom CSS, you can store it in app.css in your css/ folder


<!DOCTYPE html>
<html>
<head>
    
     <title>Foundation Template</title>
     <meta name="viewport" content="width=device-width, initial-scale=1">

     <link rel="stylesheet" href="css/app.css">    <!-- for custom css -->

     <link rel="stylesheet" href="https://cdnjs.cloudflare.com/ajax/libs/foundation/6.0.1/css/foundation.css"> 
     <script src="https://cdnjs.cloudflare.com/ajax/libs/foundation/6.0.1/js/foundation.min.js"></script>
     <script src="https://cdnjs.cloudflare.com/ajax/libs/foundation/6.0.1/js/vendor/jquery.min.js"></script>

</head>

<body>
    
     <h1>Hello, world!</h1>
	  
</body>
</html>


CDN

We'll be using CDN to access Foundation so we don't have to store a copy of it. (It's already included in the template above)

<link rel="stylesheet" href="https://cdnjs.cloudflare.com/ajax/libs/foundation/6.0.1/css/foundation.css">
